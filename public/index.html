    The section begins with the definition of a sigma field (algebra) $\textit{F}$: A sigma field is a collection of subsets of a set $\Omega$ such that the following hold: $\Omega \in \textit{F}$. if $A \in \textit{F}$, so is $A^C \in \textit{F}$. For any countable collection of sets ${A_i}_{i=0}^\infty$ in $\textit{F}$ their union is in $\textit{F}$ as well. Some observations: The empty set $\phi$ is also in F - since $\phi = \Omega^c \in \textit{F}$.
